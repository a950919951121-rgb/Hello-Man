一、操作原理

插入排序法逐個將未排序的元素插入到已排序部分中的正確位置，就像在整理撲克牌一樣。

二、虛擬代碼

procedure insertionSort( A : list of sortable items )
    n = length(A)
    for i = 1 to n-1 do
        key = A[i]
        j = i - 1
        while j >= 0 and A[j] > key do
            A[j+1] = A[j]
            j = j - 1
        end while
        A[j+1] = key
    end for
end procedure

三、複雜度分析

項目複雜度
平均時間複雜度O(n²)
最壞時間複雜度O(n²)
最好時間複雜度O(n) (當陣列已經排序好時)
空間複雜度O(1)
