一、操作原理

氣泡排序法重覆比較兩兩相鄰的元素。如果它們的順序錯誤，就將它們交換。這個過程會一直重複，直到大的元素像氣泡一樣搬到陣列的最後面。

二、虛擬代碼

procedure bubbleSort( A : list of sortable items )
    n = length(A)
    repeat
        swapped = false
        for i = 1 to n-1 inclusive do
            if A[i-1] > A[i] then
                swap( A[i-1], A[i] )
                swapped = true
            end if
        end for
        n = n - 1
    until not swapped
end procedure

三、複雜度分析

項目複雜度
平均時間複雜度O(n²)
最壞時間複雜度O(n²)
最好時間複雜度O(n) (當陣列已經排序好時)
空間複雜度O(1) (只需要額外的常數空間)
