合併排序法 (Merge Sort) 分析

1. 操作原理

合併排序採用「分治法」(Divide and Conquer) 策略：

  • 分割 (Divide)：遞迴地將目前的序列對半切分，直到子序列長度為 1。
  • 合併 (Merge)：將兩個已排序的子序列合併成一個更有序的序列。

2. 虛擬代碼 (Pseudo Code)

MERGE_SORT(A, p, r):
    if p < r:
        q = floor((p + r) / 2)
        MERGE_SORT(A, p, q)
        MERGE_SORT(A, q + 1, r)
        MERGE(A, p, q, r)

3. 複雜度分析

情況時間複雜度
最佳情況 (Best Case)O(n log n)
平均情況 (Average Case)O(n log n)
最壞情況 (Worst Case)O(n log n)
空間複雜度O(n) - 需要額外空間
