一、操作原理

合併排序法是典型的分割治領(Divide and Conquer)演算法。它會先將陣列分割成最小的單元，然後再兩兩合併並排序，直到合併成一個完整的已排序陣列。

二、虛擬代碼

procedure mergeSort( A : list of sortable items )
    if length(A) <= 1 then
        return A
    end if
    
    mid = length(A) / 2
    left = mergeSort( A[0...mid-1] )
    right = mergeSort( A[mid...end] )
    
    return merge( left, right )
end procedure

function merge( left, right )
    result = list()
    while left and right are not empty do
        if left[0] <= right[0] then
            add left[0] to result
            remove left[0] from left
        else
            add right[0] to result
            remove right[0] from right
        end if
    end while
    
    add remaining items in left or right to result
    return result
end function
