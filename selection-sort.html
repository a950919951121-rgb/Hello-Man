一、操作原理

選擇排序法會重覆地從未排序的部分中找到最小的元素，然後把它放到已排序部分的最後面。

二、虛擬代碼

procedure selectionSort( A : list of sortable items )
    n = length(A)
    for i = 0 to n-2 do
        minIndex = i
        for j = i+1 to n-1 do
            if A[j] < A[minIndex] then
                minIndex = j
            end if
        end for
        swap( A[i], A[minIndex] )
    end for
end procedure
